# Bootstrap Basics Assignment

This project is a simple Bootstrap-based registration page built with a single HTML file.

## Files

- `index.html` — main page containing a registration form, sample data table, responsive images, and a Bootstrap navbar.

## Features

- Bootstrap form with fields for:
  - First Name
  - Last Name
  - Email
  - Password
  - Terms and conditions checkbox
- Responsive layout using Bootstrap grid classes
- Form validation for required Email, Password, and checkbox fields
- Submit button styled with `btn-success`
- Responsive Bootstrap table with sample user data
- Striped and hoverable table rows for readability
- Table wrapped in `table-responsive`
- Full-width responsive image using `img-fluid` inside a `container-fluid`
- Circular image using `rounded-circle`
- Two buttons:
  - one always visible
  - one hidden on small screens with `d-none d-md-block`
- Responsive navbar with `Home`, `About`, and `Contact` links
- Navbar collapses into a hamburger menu on smaller screens

## Usage

1. Open `index.html` in a web browser.
2. Fill out the form and click `Submit` to test client-side validation.
3. Resize the browser window to see responsive behavior.

## Notes

- The Bootstrap CSS and JavaScript are loaded via CDN.
- No build process or server is required.
- This file is intended as a static Bootstrap assignment demo.
